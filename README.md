# VCB Stock Price Forecasting using N-BEATS

This repository contains the source code used in the master's thesis for forecasting VCB stock prices using Naive, ARIMA, LSTM, N-BEATS, and N-BEATSx models.

## Main Notebook

- `notebooks/VCB_Notebook_Final_vn_24_May`

## Models

- Naive baseline
- ARIMA
- LSTM
- N-BEATS
- N-BEATSx

## Dataset

The study uses VCB adjusted close price, VNINDEX, and USD/VND exchange rate data from 2015 to 2025.
